### Day 11: Bid Optimization Review

#### Step 1: Analyze Keyword-Level Performance

**Pull Keyword Report (Last 14 days):**
```
1. Navigate to: Keywords tab (all campaigns)
2. Download to Excel
3. Calculate metrics:
   - CTR = (Clicks / Impressions) * 100
   - CVR = (Conversions / Clicks) * 100
   - CPA = Spend / Conversions
   - ACOS = (Spend / Sales) * 100
```

#### Step 2: Bid Adjustment Rules

**Increase Bids (+20-30%):**
- High CVR (>8%) + Low impression share (<70%)
- High ROAS (>10x) + Spending full budget
- High CTR (>2%) + Profitable CPA

**Decrease Bids (-20-30%):**
- Low CVR (<3%) + High CPA (>$10)
- Low CTR (<0.5%)
- ROAS <5x

**Pause Keywords:**
- $50+ spend with 0 conversions
- ROAS <2.0x after $100+ spend
- CTR <0.3% after 1000+ impressions

**Implementation:**
```
1. Sort Excel by ROAS (descending)
2. Top 20% high ROAS: Increase bids 25%
3. Bottom 20% low ROAS: Decrease bids 25% or pause
4. Make changes in Amazon Ads console
5. Add notes explaining each change
```

**Verification:**

- [ ] Increased bids on top 10-15 keywords
- [ ] Decreased bids on underperformers

## Emergency Rollback Procedures

### When to Rollback

**Immediate Rollback Triggers:**
- ROAS drops below 8.0x for 3+ consecutive days
- Daily spend exceeds $175 without corresponding sales
- Cost per conversion exceeds $12 for 3+ days
- Major stockout affecting top products
- Conversion rate drops >30% (listing issue?)

---

### Rollback Procedure (2-3 hours)

#### Step 1: Pause Recent Changes

**Immediate Actions:**
```
1. Log into Amazon Ads Console
2. Navigate to: Campaign Manager
3. Filter by: "Recently Modified" (last 7 days)
4. Review each changed campaign/keyword
5. Revert to previous settings OR pause entirely
```

**Priority Order:**
1. Pause any new campaigns launched in last 7 days
2. Reduce increased budgets back to previous levels
3. Revert bid changes (undo increases)
4. Review day-of-week adjustments

---

#### Step 2: Re-Enable Paused Winners (If Needed)

**If rollback requires more spend:**
```
1. Review what was paused in Week 1
2. Evaluate if any should be re-enabled:
   - SmokeXtractor campaigns (probably not)
   - Roller Kit Tray Auto (probably not)
   - Targeting expressions (probably not)
3. If re-enabling, do so at 50% previous budget
4. Monitor for 48 hours before full restoration
```

---

#### Step 3: Root Cause Analysis

**Why Did Performance Decline?**

**Check These Factors:**
```
‚òê Product stockouts/unavailability
‚òê Competitor price changes (are we priced too high?)
‚òê Seasonal demand shifts
‚òê Listing changes (did you update anything?)
‚òê Reviews declined (negative review spike?)
‚òê Amazon algorithm changes
‚òê Budget changes were too aggressive
‚òê External factors (holiday, events)
```

**Data to Pull:**
```
1. Product page traffic and CVR (last 14 days vs previous 14)
2. Competitor pricing (manual check or tool)
3. Review score changes
4. Search ranking changes (did you drop?)
5. Advertising cost changes (did CPCs spike?)
```

---

#### Step 4: Create Recovery Plan

**If Issue is External (not your changes):**
- Maintain current setup
- Monitor daily
- Wait for market conditions to stabilize

**If Issue is Internal (your changes):**
- Revert to baseline completely
- Wait 5-7 days for performance to stabilize
- Restart optimization more conservatively:
  - Week 1: Only pause waste (don't reallocate)
  - Week 2: Small budget increases (10% vs 50%)
  - Week 3: Evaluate and proceed carefully

**Document Learnings:**
```
What went wrong: _________________________________
Why it happened: _________________________________
What we'll do differently: _______________________
```

## Appendix C: Glossary of Terms

| Term | Definition | Good Value | Bad Value |
|------|------------|------------|-----------|
| **ROAS** | Return on Ad Spend (Sales / Spend) | >10.0x | <3.0x |
| **TACOS** | Total Advertising Cost of Sales (Spend / Sales) | <10% | >20% |
| **CPA** | Cost Per Acquisition (Spend / Conversions) | <$6 | >$10 |
| **CTR** | Click-Through Rate (Clicks / Impressions) | >1.5% | <0.5% |
| **CVR** | Conversion Rate (Conversions / Clicks) | >6% | <3% |
| **CPC** | Cost Per Click (Spend / Clicks) | <$0.50 | >$0.75 |
| **ACOS** | Advertising Cost of Sales (Same as TACOS) | <10% | >20% |
| **IS** | Impression Share (Your impr / Total impr) | >70% | <40% |
| **QS** | Quality Score (Amazon's relevance metric) | 8-10 | <5 |
